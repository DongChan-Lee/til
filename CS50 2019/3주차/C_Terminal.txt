CS50 sandbox : sandbox.cs50.io
CS50 IDE : ide.cs50.io

<2주차 >
- #include <stdio.h> : 헤더 파일

[Terminal 명령어]
- clang -o hello hello.c : clang 명령어로 생성되는 output 이름 설정
- ./hello

- clang -o hello hello.c -lcs50 : cs50 라이브러리를 사용한 프로그램을 컴파일(link in)할 때
- make hello : 코드를 자동으로 컴파일링하는 간단한 명령어

단축키 Ctrl+L(혹은 clear) : 터미널 깔끔하게



<3주차>
IDE 단축키 Ctrl+C(debugger에서 다시 터미널 창으로 돌아오기; Cancel)

- help50 make 파일이름 : 컴파일 시 생기는 오류를 해석해 줌.
- (소스 코드에 직접 중지점을 지정하고 컴파일한 후에) debug50 파일명 : 오른쪽 패널을 통해 변수의 값을 확인하거나 중지점부터 Stepover 버튼을 통해 한 줄씩 코드를 실행해볼 수 있음. -> Ctrl+C(디버깅 종료)
- GDB : 자주 쓰이는 디버거 중 하나 (2. 디버깅 학습자료.pdf 참고)
